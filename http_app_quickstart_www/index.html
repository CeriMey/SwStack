SwHttpApp Quickstart

Server is running. Try these endpoints:

  • GET /api/v1/health
  • GET /api/v1/users/42?verbose=1
  • POST /api/v1/echo (raw body)
  • POST /api/v1/upload (multipart/form-data)
  • GET /api/v1/chunked
  • GET /uploads/<filename>
curl -i http://127.0.0.1:8080/api/v1/health
curl -i -X POST http://127.0.0.1:8080/api/v1/echo --data "hello"
curl -i -F "meta=alpha" -F "file=@./my.bin" http://127.0.0.1:8080/api/v1/upload
